# AORANE 🏥

  **India's Comprehensive Health Platform**

  > Privacy-first | Offline-capable | Globally scalable

  ## Platform Overview

  AORANE is a full-stack health platform built for India, consisting of:

  | App | Description | Status |
  |-----|-------------|--------|
  | 📱 Mobile App | Expo (React Native) — iOS & Android | ✅ Live |
  | 🏢 Business Portal | Web CRM for organizations | ✅ Live |
  | 🔑 Admin Panel | Super-admin control panel | ✅ Live |
  | 🖥️ API Server | Express.js + PostgreSQL | ✅ Live |

  ## Tech Stack

  - **Mobile**: Expo, React Native, Zustand, Glassmorphism UI
  - **Web**: React + Vite, TanStack Query, Wouter, Tailwind CSS
  - **Backend**: Express.js, PostgreSQL (46 tables), JWT auth
  - **Design**: Dark navy bg (#040D1C), AORANE Blue (#0077B6), Teal (#1B998B)

  ## Features

  - 🔐 Privacy-first: Sleep, Stress, Medicine tracking opt-in only
  - 🩸 Blood Emergency Network (real-time)
  - 💊 Medicine + Health tracking
  - 🍱 Food & Nutrition database
  - 🏥 Organization / Corporate wellness CRM
  - 🌐 Multi-language support (22+ Indian languages)
  - 📊 Admin panel with audit logs, feature flags, promo codes
